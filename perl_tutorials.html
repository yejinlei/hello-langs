1. Perl 简介

Perl - 实用提取与报告语言

Perl是一种功能强大的脚本语言，擅长文本处理和系统管理任务。它具有灵活的语法和丰富的模块库，广泛应用于Web开发、系统管理、网络编程等领域。

Perl 语言特点
  • 强大的正则表达式支持
  • 灵活的语法和丰富的运算符
  • 丰富的CPAN模块库
  • 跨平台兼容性
  • 擅长文本处理和字符串操作

2. Perl 基础语法

Hello World 示例
#!/usr/bin/perl
use strict;
use warnings;

print "Hello, World!\n";
变量
#!/usr/bin/perl
use strict;
use warnings;

# 标量变量
my $name = "Perl";
my $age = 30;
my $price = 19.99;

print "Name: $name\n";
print "Age: $age\n";
print "Price: $price\n";

# 数组
my @fruits = ("apple", "banana", "orange");
print "Fruits: @fruits\n";

# 哈希
my %person = (
    "name" => "John",
    "age" => 25,
    "city" => "New York"
);
print "Person: $person{name}, $person{age}, $person{city}\n";

3. Perl 控制流

条件语句
#!/usr/bin/perl
use strict;
use warnings;

my $score = 85;

if ($score >= 90) {
    print "A\n";
} elsif ($score >= 80) {
    print "B\n";
} elsif ($score >= 70) {
    print "C\n";
} else {
    print "D\n";
}
循环语句
#!/usr/bin/perl
use strict;
use warnings;

# for 循环
print "For 循环:\n";
for (my $i = 1; $i <= 5; $i++) {
    print "$i\n";
}

# while 循环
print "\nWhile 循环:\n";
my $j = 1;
while ($j <= 5) {
    print "$j\n";
    $j++;
}

# foreach 循环
print "\nForeach 循环:\n";
my @numbers = (1, 2, 3, 4, 5);
foreach my $num (@numbers) {
    print "$num\n";
}
