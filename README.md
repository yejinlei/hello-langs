# Hello 编程语言

一个多语言编程语法学习平台，支持13种主流编程语言的完整语法对比。

## 项目简介

Hello 编程语言是一个专注于编程语言语法对比学习的平台，帮助开发者快速理解和对比不同编程语言的语法特性。

## 支持的编程语言

- Python
- JavaScript
- TypeScript
- Java
- C++
- C#
- Go
- Rust
- Swift
- Kotlin
- Dart
- Ruby
- C

## 项目结构

```
hello-langs/
├── index.html          # 主页
├── tutorials.html      # 语法大全
├── lab.html           # 代码实验室
├── resources.html     # 资源中心
├── main.js            # 主要JavaScript文件
├── resources/         # 图片资源
└── README.md          # 项目说明
```

## 功能特色

- **完整语法对比**：并排显示13种语言的完整语法实现
- **在线代码实验**：支持多语言在线编辑器
- **智能学习路径**：个性化的语法学习路径

## 使用方法

1. 克隆项目到本地
2. 用浏览器打开 `index.html` 文件
3. 开始学习编程语言语法

## 部署

项目为纯静态网站，可以部署到任何静态文件服务器。
