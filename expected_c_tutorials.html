1. 变量声明

C - 静态类型语言，变量声明时需要指定类型

特点：强类型检查，变量声明必须指定类型，内存手动管理，支持多种基本类型。注意事项：注意整数溢出问题，char类型可用于存储整数，浮点数默认为double类型。

// 基本类型变量声明（C99）
int x = 10;
char name[] = "张三";
int age = 25;
float height = 1.75f;
int is_student = 1;

// 使用printf输出（C99支持可变参数宏）
printf("整数: %d, 字符串: %s\n", x, name);

2. 数据类型

C - 基本数据类型与复合类型

特点：支持整型、浮点型、字符型等基本类型，以及数组、结构体、联合体等复合类型。注意事项：不同数据类型占用内存大小可能因平台而异，使用sizeof可获取具体大小。

// 基本数据类型（C99）
char grade = 'A';
int score = 95;
float percentage = 95.5f;
double precise_value = 95.555555;
_Bool is_valid = 1;

// 复合数据类型
struct Point {
    int x, y;
}; // 结构体（C89）

typedef struct {
    int x, y;
} Point; // 类型定义（C89）
