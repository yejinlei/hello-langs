1. Lisp 简介

Lisp - 列表处理语言

Lisp是一种功能强大的函数式编程语言，以其独特的括号语法和强大的宏系统而闻名。它广泛应用于人工智能、符号计算、教育等领域。

Lisp 语言特点
  • 独特的括号语法（前缀表示法）
  • 强大的宏系统
  • 函数式编程范式
  • 动态类型系统
  • 垃圾回收机制
  • 丰富的符号处理能力

2. Lisp 基础语法

Hello World 示例
; Hello World 程序
(print "Hello, World!")

; 也可以使用 format 函数
(format t "Hello, Lisp!~%")
基本表达式
; 算术表达式（前缀表示法）
(+ 1 2 3)     ; 6
(- 10 5)      ; 5
(* 2 3 4)     ; 24
(/ 10 2)      ; 5

; 比较表达式
(= 1 1)       ; T (真)
(> 5 3)       ; T
(< 2 4)       ; T
(/= 1 2)      ; T

; 逻辑表达式
(and T T)     ; T
(or T nil)    ; T
(not T)       ; nil (假)

3. Lisp 变量和函数

变量定义
; 使用 setq 定义变量
(setq x 10)
(setq y 20)
(print (+ x y))  ; 30

; 使用 defvar 定义全局变量
(defvar *name* "Lisp")
(print *name*)

; 使用 let 定义局部变量
(let ((a 5) (b 3))
  (print (+ a b)))  ; 8
函数定义
; 使用 defun 定义函数
(defun add (a b)
  "计算两个数的和"
  (+ a b))

(print (add 5 3))  ; 8

; 递归函数
(defun factorial (n)
  "计算阶乘"
  (if (<= n 1)
      1
      (* n (factorial (- n 1)))))

(print (factorial 5))  ; 120
