1. Scala 简介

Scala - 融合面向对象与函数式编程的现代语言

Scala是一门多范式编程语言，融合了面向对象编程和函数式编程的特性。它运行在Java虚拟机上，与Java高度互操作，同时提供了更简洁、更强大的语法和功能。

Scala 语言特点
  • 融合面向对象和函数式编程范式
  • 静态类型系统，支持类型推断
  • 简洁优雅的语法设计
  • 与Java高度互操作
  • 支持模式匹配
  • 强大的集合库
  • 隐式转换和参数
  • 高阶函数和闭包

3. Scala 函数和方法

函数定义
// 标准函数定义
def add(a: Int, b: Int): Int = a + b

// 多行函数
def multiply(a: Int, b: Int): Int = {
  val result = a * b
  result
}

// 匿名函数
val subtract = (a: Int, b: Int) => a - b

// 高阶函数
def applyFunction(f: (Int, Int) => Int, x: Int, y: Int): Int = f(x, y)

val sum = applyFunction(add, 5, 3)     // 8
val diff = applyFunction(subtract, 5, 3) // 2
柯里化函数
// 柯里化函数
def curriedAdd(a: Int)(b: Int): Int = a + b

val add5 = curriedAdd(5) _   // 偏应用函数
val result = add5(3)         // 8

// 隐式参数
def greet(name: String)(implicit greeting: String): String = {
  s"$greeting, $name!"
}

implicit val defaultGreeting = "Hello"
greet("Scala") // "Hello, Scala!"

4. Scala 控制流

条件表达式
// if-else 表达式
val x = 10
val result = if (x > 5) "大于5" else "小于等于5"

// 多条件表达式
val y = 15
val category = if (y < 10) "小" else if (y < 20) "中" else "大"

// 模式匹配
val day = "Monday"
val message = day match {
  case "Monday" => "周一，开始新的一周"
  case "Friday" => "周五，周末快到了"
  case _ => "普通的一天"
}

// 类型匹配
def process(value: Any): String = value match {
  case i: Int => s"整数: $i"
  case d: Double => s"浮点数: $d"
  case s: String => s"字符串: $s"
  case _ => "其他类型"
}
循环结构
// for循环
for (i <- 1 to 5) {
  println(i)
}

// 带条件的for循环
for (i <- 1 to 10 if i % 2 == 0) {
  println(i)
}

// 多重for循环
for {
  i <- 1 to 3
  j <- 1 to 2
} println(s"i=$i, j=$j")

// while循环
var i = 1
while (i <= 5) {
  println(i)
  i += 1
}

// do-while循环
var j = 1
do {
  println(j)
  j += 1
} while (j <= 5)

5. Scala 集合

不可变集合
// 列表
val numbers = List(1, 2, 3, 4, 5)
val emptyList = Nil
val anotherList = 0 :: numbers // 前缀操作: List(0, 1, 2, 3, 4, 5)

// 集合
val uniqueNumbers = Set(1, 2, 3, 2, 1) // Set(1, 2, 3)

// 元组
val tuple = (1, "Hello", 3.14) // Tuple3[Int, String, Double]
val first = tuple._1 // 1
val second = tuple._2 // "Hello"

// 映射
val map = Map("name" -> "Scala", "version" -> "3.0")
val name = map("name") // "Scala"
val optionalVersion = map.get("version") // Some("3.0")
集合操作
// 转换操作
val doubled = numbers.map(_ * 2) // List(2, 4, 6, 8, 10)
val evenNumbers = numbers.filter(_ % 2 == 0) // List(2, 4)
val summed = numbers.reduce(_ + _) // 15
val sortedDesc = numbers.sorted(Ordering[Int].reverse) // List(5, 4, 3, 2, 1)

// 折叠操作
val sumWithStart = numbers.fold(0)(_ + _) // 15
val sumLeft = numbers.foldLeft(0)(_ + _) // 15
val sumRight = numbers.foldRight(0)(_ + _) // 15

// 扁平化
val nested = List(List(1, 2), List(3, 4))
val flattened = nested.flatten // List(1, 2, 3, 4)

// 扁平映射
val flatMapped = nested.flatMap(list => list.map(_ * 2)) // List(2, 4, 6, 8)

6. Scala 面向对象编程

类和对象
// 类定义
class Person(var name: String, var age: Int) {
  // 辅助构造函数
  def this(name: String) = this(name, 0)
  
  // 方法
  def greet(): String = s"Hello, my name is $name and I'm $age years old."
}

// 对象实例化
val person1 = new Person("Alice", 30)
val person2 = new Person("Bob")

person1.age = 31 // 修改属性
println(person1.greet()) // "Hello, my name is Alice and I'm 31 years old."

// 单例对象
object MyApp {
  def main(args: Array[String]): Unit = {
    println("Hello from singleton object!")
  }
}
继承和特质
// 特质（类似接口）
trait Greeting {
  def greet(): String
}

// 抽象类
abstract class Animal {
  def sound(): String
}

// 类继承抽象类并实现特质
class Dog extends Animal with Greeting {
  override def sound(): String = "Woof!"
  override def greet(): String = "Hello, I'm a dog!"
}

val dog = new Dog()
println(dog.sound()) // "Woof!"
println(dog.greet()) // "Hello, I'm a dog!"

// 密封类（只能在当前文件中被继承）
sealed trait Shape
case class Circle(radius: Double) extends Shape
case class Rectangle(width: Double, height: Double) extends Shape

// 模式匹配使用密封类
def area(shape: Shape): Double = shape match {
  case Circle(r) => Math.PI * r * r
  case Rectangle(w, h) => w * h
}
