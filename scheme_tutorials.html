1. Scheme 简介

Scheme - 简洁优雅的函数式编程语言

Scheme是Lisp编程语言家族的一个方言，以其简洁的语法、强大的函数式编程能力和数学般的优雅而著称。它是计算机科学教育的重要语言，被广泛用于编程语言理论和算法教学。

Scheme 语言特点
  • 极简的语法设计（仅包含少数核心结构）
  • 纯粹的函数式编程范式
  • 强大的宏系统（卫生宏）
  • 尾递归优化支持
  • 动态类型系统
  • 垃圾回收机制
  • 闭包和第一类函数

2. Scheme 基本语法

简洁而强大的表达式系统

Scheme使用极简的语法，几乎所有的操作都通过表达式实现。基本语法包括表达式、注释和简单的数据类型。

Hello World 示例
; Hello World 程序
(display "Hello, World!")
(newline)

; 也可以使用简单的表达式
"Hello, Scheme!"
基本表达式
; 算术表达式（前缀表示法）
(+ 1 2 3)     ; 6
(- 10 5)      ; 5
(* 2 3 4)     ; 24
(/ 10 2)      ; 5
(remainder 10 3) ; 1

; 比较表达式
(= 1 1)       ; #t (真)
(> 5 3)       ; #t
(< 2 4)       ; #t
(not (= 1 2)) ; #t
(and #t #t)   ; #t
(or #t #f)    ; #t
(not #t)      ; #f (假)

3. Scheme 变量与函数

函数式编程的核心元素

Scheme支持变量的定义和函数的创建，函数是Scheme中的一等公民，可以作为参数传递、作为返回值和存储在数据结构中。

变量定义
; 使用 define 定义变量
(define x 10)
(define y 20)
(display (+ x y)) ; 30
(newline)

; 使用 let 定义局部变量
(let ((a 5) (b 3))
  (display (+ a b)) ; 8
  (newline))

; let* 允许变量之间相互引用
(let* ((a 2)
       (b (* a 3))) ; b = 6
  (display b)
  (newline))
函数定义
; 使用 define 定义函数
(define (add a b)
  "计算两个数的和"
  (+ a b))

(display (add 5 3)) ; 8
(newline)

; 尾递归函数（会被优化为循环）
(define (factorial n)
  "计算阶乘（尾递归版本）"
  (define (fact-iter product counter max-count)
    (if (> counter max-count)
        product
        (fact-iter (* product counter) (+ counter 1) max-count)))
  (fact-iter 1 1 n))

(display (factorial 5)) ; 120
(newline)

; 高阶函数 - 返回函数的函数
(define (make-adder x)
  (lambda (y) (+ x y)))

(define add5 (make-adder 5))
(display (add5 3)) ; 8
(newline)

4. Scheme 条件与循环

控制程序的执行流程

Scheme提供了条件表达式来控制程序的分支，以及通过递归来实现循环功能，这是函数式编程的典型特征。

条件表达式
; if 条件表达式
(define (abs x)
  (if (< x 0)
      (- x)
      x))

(display (abs -5)) ; 5
(newline)

; cond 多分支条件
(define (grade score)
  (cond ((>= score 90) "A")
        ((>= score 80) "B")
        ((>= score 70) "C")
        ((>= score 60) "D")
        ((else) "F")))

(display (grade 85)) ; B
(newline)
循环结构
; 使用递归实现循环
(define (countdown n)
  (if (> n 0)
      (begin
        (display n)
        (newline)
        (countdown (- n 1)))
      (display "Blast off!")
      (newline)))

(countdown 3)
; 输出:
; 3
; 2
; 1
; Blast off!

; 使用 do 循环
(do ((i 0 (+ i 1)))
    ((>= i 3))
  (display i)
  (newline))
; 输出:
; 0
; 1
; 2

5. Scheme 数据结构

灵活多样的数据表示方式

Scheme提供了多种数据结构来表示和组织数据，包括列表、字符串、符号和向量等，这些数据结构可以组合使用来构建复杂的数据模型。

列表操作
; 创建列表
(define numbers (list 1 2 3 4 5))
(define empty-list '())

; 列表操作
(display (car numbers))   ; 1 (第一个元素)
(newline)
(display (cdr numbers))   ; (2 3 4 5) (剩余元素)
(newline)
(display (cons 0 numbers)) ; (0 1 2 3 4 5) (添加到开头)
(newline)
(display (append numbers (list 6 7))) ; (1 2 3 4 5 6 7) (连接列表)
(newline)
(display (length numbers)) ; 5 (列表长度)
(newline)
(display (reverse numbers)) ; (5 4 3 2 1) (反转列表)
(newline)
其他数据类型
; 字符串
(define greeting "Hello, Scheme!")
(display greeting)
(newline)
(display (string-length greeting)) ; 14
(newline)

; 符号
(define symbol 'my-symbol)
(display symbol)
(newline)

; 向量（固定大小的数组）
(define vector1 #(1 2 3 4 5))
(display (vector-ref vector1 2)) ; 3
(newline)
(vector-set! vector1 2 10)
(display (vector-ref vector1 2)) ; 10
(newline)
