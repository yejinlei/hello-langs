1. HTML基础

什么是HTML？

HTML（HyperText Markup Language）是用于创建网页的标准标记语言。它描述了网页的结构，并使用标记标签来定义内容的类型和格式。

HTML文档结构

一个基本的HTML文档包含以下结构：

<!DOCTYPE html>
<html lang="zh-CN">
<head>
<meta charset="UTF-8">
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<title>我的第一个HTML页面</title>
</head>
<body>
<h1>Hello, World!</h1>
<p>这是我的第一个HTML段落。</p>
</body>
</html>

2. HTML元素和标签

什么是HTML元素？

HTML元素是HTML文档的基本组成部分，由开始标签、内容和结束标签组成。

<tagname>内容</tagname>

块级元素

块级元素在页面上占据一整行，默认情况下会在前后创建换行。

<h1>标题元素</h1>
<p>段落元素</p>
<div>容器元素</div>
<ul>无序列表</ul>
<ol>有序列表</ol>
<li>列表项</li>

行内元素

行内元素只占据其内容所需的空间，不会创建换行。

<a>链接元素</a>
<span>行内容器</span>
<strong>粗体文本</strong>
<em>斜体文本</em>
<img>图片元素
<input>输入元素

空元素

空元素没有内容，也不需要结束标签。

<br>换行元素
<hr>水平分隔线
<img src="image.jpg" alt="图片">
<input type="text" placeholder="输入文本">

3. HTML属性

什么是HTML属性？

HTML属性提供了关于HTML元素的额外信息，通常在开始标签中定义。

<tagname attribute="value">内容</tagname>

全局属性

全局属性可以应用于所有HTML元素。

<div id="my-id">唯一标识</div>
<p class="my-class">样式类</p>
<span style="color: red;">内联样式</span>
<img alt="替代文本">
<a href="https://example.com">链接</a>

id属性

id属性为HTML元素提供唯一标识，在同一文档中不能重复。

<div id="header">
<h1>页面标题</h1>
</div>

class属性

class属性用于为HTML元素定义一个或多个类名，可在同一文档中重复使用。

<p class="text-red">红色文本</p>
<p class="text-red bold">红色粗体文本</p>

4. HTML表单

什么是HTML表单？

HTML表单用于收集用户输入，允许用户将数据发送到服务器。

<form action="/submit" method="post">
<input type="submit" value="提交">
</form>

表单元素

HTML表单包含多种输入元素。

<!-- 文本输入 -->
<input type="text" name="username" placeholder="用户名">

<!-- 密码输入 -->
<input type="password" name="password" placeholder="密码">

<!-- 单选按钮 -->
<input type="radio" name="gender" value="male"> 男
<input type="radio" name="gender" value="female"> 女

<!-- 复选框 -->
<input type="checkbox" name="hobby" value="reading"> 阅读
<input type="checkbox" name="hobby" value="sports"> 运动

<!-- 下拉列表 -->
<select name="country">
<option value="china">中国</option>
<option value="usa">美国</option>
</select>

<!-- 文本区域 -->
<textarea name="message" rows="4" cols="50">请输入消息...</textarea>

<!-- 提交按钮 -->
<input type="submit" value="提交">

标签关联

使用label标签可以将文本与表单元素关联，提高可访问性。

<label for="username">用户名：</label>
<input type="text" id="username" name="username">

表单验证

HTML5提供了内置的表单验证功能。

<!-- 必填字段 -->
<input type="text" name="email" required>

<!-- 邮箱验证 -->
<input type="email" name="email">

<!-- 数字验证 -->
<input type="number" name="age" min="18" max="100">

<!-- 正则表达式验证 -->
<input type="text" name="phone" pattern="[0-9]{11}">

5. HTML语义化标签

什么是语义化标签？

语义化标签是指能够清晰地描述其内容含义的HTML标签，而不仅仅是定义内容的显示方式。

使用语义化标签可以提高页面的可访问性、SEO和可维护性。

常用语义化标签

<header>页面或区块头部</header>
<nav>导航链接区域</nav>
<main>页面主要内容</main>
<section>文档中的一个区块</section>
<article>独立的内容单元</article>
<aside>侧边栏或相关内容</aside>
<footer>页面或区块底部</footer>
<figure>媒体内容及其标题</figure>
<figcaption>媒体内容的标题</figcaption>
<time>日期或时间</time>

<!DOCTYPE html>

</head>

<main>
<section>
<h2>文章列表</h2>

</footer>

语义化标签的优势

  • 提高页面的可访问性，帮助屏幕阅读器等辅助技术理解页面结构
  • 改善SEO，搜索引擎可以更好地理解页面内容
  • 增强代码的可读性和可维护性
  • 使页面结构更加清晰，便于团队协作开发

6. CSS基础

什么是CSS？

CSS（Cascading Style Sheets）是用于描述HTML文档样式的语言，包括颜色、布局、字体等。

CSS语法

CSS规则由选择器和声明块组成。

selector {
property: value;
property: value;
}

CSS引入方式

1. 内联样式

直接在HTML元素中使用style属性。

<p style="color: red; font-size: 16px;">红色文本</p>
2. 内部样式表

在HTML文档的head部分使用style标签。

<head>
<style>
p {
color: red;
font-size: 16px;
}
</style>
</head>
3. 外部样式表

创建独立的CSS文件，然后在HTML中引用。

/* style.css */
p {
color: red;
font-size: 16px;
}
<head>
<link rel="stylesheet" href="style.css">
</head>

CSS注释

使用/* */来添加注释，注释不会影响样式显示。

/* 这是一个CSS注释 */
p {
color: red; /* 设置文本颜色 */
}

CSS选择器优先级

CSS选择器具有不同的优先级，决定了哪些样式会被应用。

  • 内联样式 (style属性) - 优先级最高
  • ID选择器 (#id)
  • 类选择器 (.class)、属性选择器 ([attribute])、伪类 (:hover)
  • 元素选择器 (element)、伪元素 (::before)

7. CSS选择器

什么是CSS选择器？

CSS选择器用于选择要设置样式的HTML元素。

基本选择器

1. 元素选择器

选择所有指定类型的HTML元素。

p {
color: red;
}
2. ID选择器

选择具有指定ID的HTML元素。

#my-id {
font-size: 20px;
}
3. 类选择器

选择具有指定类名的HTML元素。

.my-class {
background-color: yellow;
}
4. 通用选择器

选择所有HTML元素。

* {
margin: 0;
padding: 0;
}

组合选择器

1. 后代选择器

选择指定元素的所有后代元素。

div p {
color: blue;
}
2. 子选择器

选择指定元素的直接子元素。

div > p {
color: green;
}
3. 相邻兄弟选择器

选择紧接在指定元素之后的兄弟元素。

p + div {
margin-top: 20px;
}
4. 通用兄弟选择器

选择指定元素之后的所有兄弟元素。

p ~ div {
color: purple;
}

属性选择器

根据元素的属性或属性值选择元素。

/* 选择具有指定属性的元素 */
[target] {
color: red;
}

/* 选择具有指定属性和值的元素 */
[target="_blank"] {
color: blue;
}

/* 选择属性值以指定值开头的元素 */
[href^="https"] {
color: green;
}

/* 选择属性值以指定值结尾的元素 */
[href$=".pdf"] {
color: orange;
}

伪类选择器

选择处于特定状态的元素。

/* 鼠标悬停状态 */
a:hover {
color: red;
}

/* 已访问链接 */
a:visited {
color: purple;
}

/* 第一个子元素 */
p:first-child {
font-weight: bold;
}

/* 偶数位置的元素 */
li:nth-child(even) {
background-color: #f2f2f2;
}

伪元素选择器

向元素的特定部分添加样式。

/* 在元素内容前添加内容 */
p::before {
content: "★";
}

/* 在元素内容后添加内容 */
p::after {
content: "✨";
}

/* 选择元素的第一行 */
p::first-line {
font-weight: bold;
}

8. CSS盒模型

什么是CSS盒模型？

CSS盒模型是CSS布局的基础，它定义了元素在页面上的尺寸和边距。

盒模型的组成部分

CSS盒模型由以下几个部分组成：

  • 内容区（Content）：元素的实际内容，显示文本和图像
  • 内边距（Padding）：内容区与边框之间的空间
  • 边框（Border）：围绕内容区和内边距的边框
  • 外边距（Margin）：元素与其他元素之间的空间

盒模型尺寸

元素的总宽度和总高度可以通过以下公式计算：

总宽度 = width + padding-left + padding-right + border-left + border-right + margin-left + margin-right
总高度 = height + padding-top + padding-bottom + border-top + border-bottom + margin-top + margin-bottom

盒模型属性

1. 内容区属性
div {
width: 300px;
height: 200px;
min-width: 200px;
max-width: 500px;
min-height: 100px;
max-height: 400px;
}
2. 内边距属性
div {
padding: 20px; /* 四个方向都为20px */
}

div {
padding: 10px 20px; /* 上下10px，左右20px */
}

div {
padding: 10px 20px 30px; /* 上10px，左右20px，下30px */
}

div {
padding: 10px 20px 30px 40px; /* 上10px，右20px，下30px，左40px */
}

div {
padding-top: 10px;
padding-right: 20px;
padding-bottom: 30px;
padding-left: 40px;
}
3. 边框属性
div {
border: 2px solid #ccc; /* 边框宽度、样式、颜色 */
}

div {
border-width: 2px;
border-style: solid; /* solid, dotted, dashed, double, groove, ridge, inset, outset */
border-color: #ccc;
}

div {
border-top: 2px solid red;
border-right: 2px dashed blue;
border-bottom: 2px dotted green;
border-left: 2px double yellow;
}
4. 外边距属性
div {
margin: 20px; /* 四个方向都为20px */
}

div {
margin: 10px 20px; /* 上下10px，左右20px */
}

div {
margin-top: 10px;
margin-right: 20px;
margin-bottom: 30px;
margin-left: 40px;
}

box-sizing属性

box-sizing属性用于控制如何计算元素的总宽度和高度。

1. content-box（默认值）

宽度和高度仅包括内容区，不包括内边距、边框和外边距。

div {
box-sizing: content-box;
width: 300px;
padding: 20px;
border: 5px solid #ccc;
/* 总宽度 = 300px + 20px*2 + 5px*2 = 350px */
}
2. border-box

宽度和高度包括内容区、内边距和边框，但不包括外边距。

div {
box-sizing: border-box;
width: 300px;
padding: 20px;
border: 5px solid #ccc;
/* 总宽度 = 300px，内容区宽度 = 300px - 20px*2 - 5px*2 = 250px */
}
3. 全局重置box-sizing

通常在CSS中使用以下代码将所有元素的box-sizing重置为border-box，以便更容易地进行布局。

*, *::before, *::after {
box-sizing: border-box;
}

9. CSS定位

什么是CSS定位？

CSS定位（Positioning）允许我们精确控制元素在页面上的位置。

position属性的取值

position属性有以下几个取值：

1. 静态定位（static）

默认的定位方式，元素按照正常的文档流排列，不受到top、bottom、left、right和z-index属性的影响。

div {
position: static;
}
2. 相对定位（relative）

元素相对于其正常位置进行定位，可以使用top、bottom、left、right属性来调整位置。

div {
position: relative;
top: 10px; /* 向下移动10px */
left: 20px; /* 向右移动20px */
}
3. 绝对定位（absolute）

元素相对于最近的定位祖先元素（position值不是static的元素）进行定位，如果没有这样的祖先元素，则相对于文档的body元素。

.parent {
position: relative; /* 父元素设置为相对定位 */
}

.child {
position: absolute;
top: 0;
right: 0; /* 相对于父元素右上角定位 */
}
4. 固定定位（fixed）

元素相对于浏览器窗口进行定位，即使页面滚动，元素也会保持在固定位置。

.header {
position: fixed;
top: 0;
left: 0;
width: 100%;
background-color: #333;
}
5. 粘性定位（sticky）

元素在正常文档流中排列，当滚动到特定位置时，会固定在视口中。

.sticky-element {
position: sticky;
top: 0; /* 当滚动到顶部时固定 */
background-color: #f1f1f1;
}

z-index属性

z-index属性用于控制定位元素的堆叠顺序，数值越大，元素越靠上。

.element-1 {
position: absolute;
z-index: 1;
}

.element-2 {
position: absolute;
z-index: 2; /* 会覆盖element-1 */
}

定位实例：创建一个悬浮按钮

.floating-button {
position: fixed;
bottom: 20px;
right: 20px;
width: 60px;
height: 60px;
border-radius: 50%;
background-color: #007bff;
color: white;
font-size: 24px;
border: none;
cursor: pointer;
}

10. CSS浮动和清除

什么是CSS浮动？

CSS浮动（Floating）允许元素向左或向右移动，直到它的外边缘碰到包含它的框或另一个浮动元素的边缘。

float属性

float属性用于设置元素的浮动方向，有以下几个取值：

1. left

元素向左浮动。

img {
float: left;
margin-right: 10px;
}
2. right

元素向右浮动。

img {
float: right;
margin-left: 10px;
}
3. none

默认值，元素不浮动，按照正常的文档流排列。

div {
float: none;
}
4. inherit

继承父元素的float属性。

div {
float: inherit;
}

浮动的效果

当元素浮动时，它会脱离正常的文档流，向左或向右移动，直到碰到包含它的容器或另一个浮动元素的边缘。

1. 图文混排
<div class="container">
<img src="image.jpg" alt="示例图片">
<p>这是一段示例文本，当图片浮动时，文本会围绕在图片周围。</p>
</div>

img {
float: left;
margin-right: 15px;
}
2. 多列布局
<div class="column left">左侧列</div>
<div class="column right">右侧列</div>

.column {
width: 50%;
padding: 15px;
}

.left {
float: left;
}

.right {
float: right;
}

清除浮动

当元素浮动时，它会脱离文档流，这可能会导致父元素的高度塌陷。我们需要清除浮动来解决这个问题。

1. clear属性

clear属性用于指定元素的哪一侧不允许有浮动元素。

.clear-left {
clear: left; /* 左侧不允许有浮动元素 */
}

.clear-right {
clear: right; /* 右侧不允许有浮动元素 */
}

.clear-both {
clear: both; /* 两侧都不允许有浮动元素 */
}
2. overflow方法

将父元素的overflow属性设置为auto或hidden，可以清除浮动。

.container {
overflow: auto; /* 清除浮动 */
}
3. clearfix技巧

使用伪元素清除浮动，这是一种常用的清除浮动的方法。

.clearfix::after {
content: "";
display: table;
clear: both;
}

浮动布局实例

<div class="header">Header</div>
<div class="container clearfix">
<div class="nav">Navigation</div>
<div class="content">Content</div>
</div>
<div class="footer">Footer</div>

.nav {
float: left;
width: 25%;
}

.content {
float: right;
width: 75%;
}

/* 使用clearfix清除浮动 */
.clearfix::after {
content: "";
display: table;
clear: both;
}

11. CSS Flexbox

什么是CSS Flexbox？

CSS Flexbox（弹性盒子布局）是一种一维布局模型，用于更灵活地排列元素。它允许容器内的元素自动分配空间，适应不同屏幕尺寸。

Flexbox的基本概念

Flexbox布局由容器和项目组成：

  • 容器（Container）：应用了display: flex的父元素
  • 项目（Items）：容器内的直接子元素
  • 主轴（Main Axis）：默认是水平方向，由flex-direction属性控制
  • 交叉轴（Cross Axis）：与主轴垂直的方向

容器属性

1. display

将元素设置为Flex容器。

.container {
display: flex; /* 块级Flex容器 */
}

.container {
display: inline-flex; /* 行内Flex容器 */
}
2. flex-direction

控制主轴的方向。

.container {
flex-direction: row; /* 默认值，水平从左到右 */
}

.container {
flex-direction: row-reverse; /* 水平从右到左 */
}

.container {
flex-direction: column; /* 垂直从上到下 */
}

.container {
flex-direction: column-reverse; /* 垂直从下到上 */
}
3. flex-wrap

控制项目是否换行。

.container {
flex-wrap: nowrap; /* 默认值，不换行 */
}

.container {
flex-wrap: wrap; /* 换行 */
}

.container {
flex-wrap: wrap-reverse; /* 反向换行 */
}
4. flex-flow

flex-direction和flex-wrap的简写属性。

.container {
flex-flow: row wrap; /* 等同于flex-direction: row; flex-wrap: wrap; */
}
5. justify-content

控制项目在主轴上的对齐方式。

.container {
justify-content: flex-start; /* 默认值，从起点对齐 */
}

.container {
justify-content: flex-end; /* 从终点对齐 */
}

.container {
justify-content: center; /* 居中对齐 */
}

.container {
justify-content: space-between; /* 两端对齐，项目之间有相等间距 */
}

.container {
justify-content: space-around; /* 每个项目两侧的间距相等 */
}

.container {
justify-content: space-evenly; /* 每个项目之间的间距相等 */
}
6. align-items

控制项目在交叉轴上的对齐方式。

.container {
align-items: stretch; /* 默认值，拉伸填满容器 */
}

.container {
align-items: flex-start; /* 交叉轴起点对齐 */
}

.container {
align-items: flex-end; /* 交叉轴终点对齐 */
}

.container {
align-items: center; /* 交叉轴居中对齐 */
}

.container {
align-items: baseline; /* 基线对齐 */
}
7. align-content

控制多行项目在交叉轴上的对齐方式（仅当项目换行时有效）。

.container {
align-content: stretch; /* 默认值，拉伸填满容器 */
}

.container {
align-content: flex-start; /* 交叉轴起点对齐 */
}

.container {

12. CSS Grid

什么是CSS Grid？

CSS Grid（网格布局）是一种二维布局模型，用于创建复杂的网格结构，允许同时控制行和列。

Grid的基本概念

Grid布局由容器和项目组成：

  • 容器（Container）：应用了display: grid的父元素
  • 项目（Items）：容器内的直接子元素
  • 网格线（Grid Lines）：构成网格结构的水平线和垂直线
  • 网格轨道（Grid Tracks）：两条相邻网格线之间的空间（行或列）
  • 网格单元格（Grid Cell）：两条相邻行线和两条相邻列线之间的空间
  • 网格区域（Grid Area）：由多个网格单元格组成的矩形区域

3. grid-template-rows

13. CSS响应式设计

什么是响应式设计？

响应式设计是一种设计理念，确保网站在不同设备和屏幕尺寸上都能提供良好的用户体验。

视口（Viewport）

视口是用户在浏览器中看到的网页区域。为了实现响应式设计，我们需要在HTML文档的头部添加视口元标签。

<meta name="viewport" content="width=device-width, initial-scale=1.0">

媒体查询（Media Queries）

媒体查询是响应式设计的核心，它允许我们根据设备的特性来应用不同的CSS样式。

1. 媒体查询的基本语法
@media (条件) {
/* 在条件满足时应用的CSS样式 */
}
2. 常用媒体特性
  • width：视口宽度
  • height：视口高度
  • orientation：设备方向（portrait或landscape）
  • min-width：最小视口宽度
  • max-width：最大视口宽度
3. 媒体查询实例
/* 当屏幕宽度小于600px时 */
@media (max-width: 600px) {
body {
font-size: 14px;
}
}

/* 当屏幕宽度在600px到900px之间时 */
@media (min-width: 600px) and (max-width: 900px) {
.container {
width: 80%;
}
}

/* 当设备处于横向模式时 */
@media (orientation: landscape) {
.sidebar {
width: 200px;
}
}

响应式布局技术

1. 流动布局（Fluid Layout）

使用百分比而不是固定像素来设置宽度，使元素能够根据容器大小自动调整。

.container {
width: 80%;
margin: 0 auto;
}
2. 弹性布局（Flexbox）

使用Flexbox可以轻松创建响应式布局，使元素能够灵活地排列和调整大小。

.flex-container {
display: flex;
flex-wrap: wrap;
}
3. 网格布局（Grid）

Grid布局是一种二维布局系统，可以轻松创建复杂的响应式布局。

.grid-container {
display: grid;
grid-template-columns: repeat(auto-fit, minmax(200px, 1fr));
}

响应式图片

在响应式设计中，我们需要确保图片能够根据屏幕尺寸自动调整大小。

1. 使用max-width属性
img {
max-width: 100%;
height: auto;
}
2. 使用picture元素
<picture>
<source media="(max-width: 600px)" srcset="small.jpg">
<source media="(max-width: 900px)" srcset="medium.jpg">
<img src="large.jpg" alt="响应式图片">
</picture>

响应式字体

在响应式设计中，我们可以使用相对单位和媒体查询来调整字体大小。

1. 使用相对单位
body {
font-size: 16px;
}

h1 {
font-size: 2rem; /* 相对于根元素的字体大小 */
}
2. 使用媒体查询调整字体大小
h1 {
font-size: 3rem;
}

@media (max-width: 768px) {
h1 {
font-size: 2rem;
}
}

@media (max-width: 480px) {
h1 {
font-size: 1.5rem;
}
}

14. CSS动画和过渡

CSS过渡（Transitions）

CSS过渡允许我们在元素从一种状态变换到另一种状态时，平滑地改变元素的样式。

1. 过渡的基本属性
.element {
transition-property: width; /* 指定要过渡的CSS属性 */
transition-duration: 1s; /* 过渡持续时间 */
transition-timing-function: ease; /* 过渡时间函数 */
transition-delay: 0.5s; /* 过渡延迟时间 */
}
2. transition-property

指定要应用过渡效果的CSS属性名称。可以使用`all`关键字表示所有可过渡的属性。

.element {
transition-property: width, height, background-color;
}

.element {
transition-property: all;
}
3. transition-duration

指定过渡效果的持续时间，单位可以是秒(s)或毫秒(ms)。

.element {
transition-duration: 2s;
}

.element {
transition-duration: 500ms;
}
4. transition-timing-function

指定过渡效果的时间函数，控制过渡的速度变化。

  • linear：匀速过渡
  • ease：默认值，先慢后快再慢
  • ease-in：慢进
  • ease-out：慢出
  • ease-in-out：慢进慢出
  • cubic-bezier(n,n,n,n)：自定义贝塞尔曲线
.element {
transition-timing-function: ease-in-out;
}
5. transition-delay

指定过渡效果开始前的延迟时间。

.element {
transition-delay: 1s;
}
6. transition 简写属性

可以使用`transition`简写属性一次性设置所有过渡属性。

.element {
transition: width 2s ease-in-out 0.5s;
}

.element {
transition: all 1s ease;
}
7. 过渡实例
.button {
padding: 10px 20px
background-color: #3498db
color: white
border: none
border-radius: 5px
transition: all 0.3s ease
}

.button:hover {
background-color: #2980b9
transform: scale(1.1)
}

CSS动画（Animations）

CSS动画允许我们创建更复杂的动画效果，不需要JavaScript。

1. @keyframes 规则

使用`@keyframes`规则定义动画序列，指定动画在不同阶段的样式。

@keyframes slideIn {
from {
transform: translateX(-100%)
}

to {
transform: translateX(0)
}
}

/* 使用百分比定义多个关键帧 */
@keyframes bounce {
0% {
transform: translateY(0)
}
50% {
transform: translateY(-50px)
}
100% {
transform: translateY(0)
}
}
2. 动画属性
.element {
animation-name: bounce; /* 指定动画名称 */
animation-duration: 2s; /* 动画持续时间 */
animation-timing-function: ease; /* 动画时间函数 */
animation-delay: 0.5s; /* 动画延迟时间 */
animation-iteration-count: infinite; /* 动画播放次数 */
animation-direction: alternate; /* 动画播放方向 */
animation-fill-mode: both; /* 动画填充模式 */
animation-play-state: running; /* 动画播放状态 */
}
3. animation-iteration-count

指定动画播放的次数，可以是数字或`infinite`（无限循环）。

.element {
animation-iteration-count: 3;
}

.element {
animation-iteration-count: infinite;
}
4. animation-direction

指定动画播放的方向。

  • normal：默认值，正向播放
  • reverse：反向播放
  • alternate：正向和反向交替播放
  • alternate-reverse：反向和正向交替播放
.element {
animation-direction: alternate;
}
5. animation-fill-mode

指定动画在播放前后如何应用样式。

  • none：默认值，动画播放前后不应用任何样式
  • forwards：保持动画结束时的样式
  • backwards：应用动画开始前的样式
  • both：同时应用forwards和backwards的效果
.element {
animation-fill-mode: forwards;
}
6. animation-play-state

指定动画是运行还是暂停。

  • running：默认值，动画运行
  • paused：动画暂停
.element:hover {
animation-play-state: paused;
}
7. animation 简写属性

可以使用`animation`简写属性一次性设置所有动画属性。

.element {
animation: bounce 2s ease 0.5s infinite alternate both
}

15. CSS变量

什么是CSS变量？

CSS变量（也称为自定义属性）是CSS中的一个强大功能，允许我们定义可重用的值，这些值可以在整个CSS文件中使用。

CSS变量的基本语法

1. 变量的声明

CSS变量以`--`开头，通常在`:root`伪类中定义，以便在整个文档中使用。

:root {
--primary-color: #3498db;
--secondary-color: #2ecc71;
--font-size: 16px;
--spacing: 20px;
}
2. 变量的使用

使用`var()`函数来引用CSS变量。

body {
font-size: var(--font-size)
color: var(--primary-color)
}

.button {
background-color: var(--primary-color)
padding: var(--spacing) var(--spacing) * 2
}
3. 变量的默认值

`var()`函数可以接受第二个参数作为默认值，当变量未定义时使用。

.box {
width: var(--box-width, 300px)
height: var(--box-height, 200px)
}

CSS变量的作用域

CSS变量的作用域由声明它们的选择器决定。

1. 全局作用域

在`:root`或`html`选择器中声明的变量具有全局作用域，可以在整个文档中使用。

:root {
--global-color: red
}

.any-element {
color: var(--global-color) /* 有效 */
}
2. 局部作用域

在特定元素中声明的变量只在该元素及其子元素中有效。

.container {
--local-color: blue
}

.container p {
color: var(--local-color) /* 有效 */
}

.outside {
color: var(--local-color) /* 无效，变量不在作用域内 */
}

CSS变量的继承

CSS变量会继承，子元素可以访问父元素中定义的变量。

.parent {
--parent-color: green
}

.child {
color: var(--parent-color) /* 继承自父元素 */
}

CSS变量的修改

CSS变量可以在不同的选择器中被重新定义，从而实现样式的覆盖。

:root {
--primary-color: #3498db
}

.button {
background-color: var(--primary-color)
}

.button.danger {
--primary-color: #e74c3c
}

CSS变量与媒体查询

CSS变量可以在媒体查询中被修改，从而实现响应式设计。

:root {
--font-size: 16px
--spacing: 20px
}

@media (max-width: 768px) {
:root {
--font-size: 14px
--spacing: 15px
}
}

CSS变量与JavaScript

JavaScript可以访问和修改CSS变量，这使得我们可以创建动态的样式效果。

/* 获取CSS变量的值 */
// 在JavaScript中
const root = getComputedStyle(document.documentElement);
const primaryColor = root.getPropertyValue('--primary-color');
console.log(primaryColor); // 输出: #3498db

/* 设置CSS变量的值 */
document.documentElement.style.setProperty('--primary-color', '#e74c3c');

/* 修改特定元素的CSS变量 */
const button = document.querySelector('.button');
button.style.setProperty('--primary-color', '#2ecc71');

CSS变量的实际应用

1. 主题切换

6. 本地存储

HTML5提供了两种本地存储机制：localStorage和sessionStorage。

localStorage

localStorage用于长期存储数据，数据不会因为浏览器关闭而丢失。

<script>
// 存储数据
localStorage.setItem('username', '张三');
localStorage.setItem('email', 'zhangsan@example.com');

// 获取数据
const username = localStorage.getItem('username');
const email = localStorage.getItem('email');

// 删除数据
localStorage.removeItem('email');

// 清空所有数据
localStorage.clear();
</script>
sessionStorage

sessionStorage用于临时存储数据，数据会在浏览器关闭时丢失。

<script>
// 存储数据
sessionStorage.setItem('tempData', '这是临时数据');

// 获取数据
const tempData = sessionStorage.getItem('tempData');

// 删除数据
sessionStorage.removeItem('tempData');

// 清空所有数据
sessionStorage.clear();
</script>

console.log('计算结果:', e.data);
};
</script>

// worker.js
self.onmessage = (e) => {
const num = e.data;
// 执行耗时计算
let result = 0;
for (let i = 0; i < num; i++) {
result += i;
}

// 返回结果
self.postMessage(result);
};

CSS提供了多种函数来进行值的计算和操作，使样式更加灵活和动态。

用于在CSS中进行数值计算，可以混合使用不同的单位。

}

.box {

clamp()函数

用于创建一个在最小值和最大值之间动态变化的值。

.box {

min()和max()函数

min()返回多个值中的最小值，max()返回多个值中的最大值。

.container {

.box {
