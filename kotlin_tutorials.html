1. Kotlin 简介

Kotlin - 现代化的跨平台编程语言

Kotlin是由JetBrains开发的一种静态类型、跨平台的编程语言，它运行在Java虚拟机(JVM)上，也可以编译为JavaScript或本地代码。Kotlin结合了面向对象编程和函数式编程的特性，以其简洁性、安全性和与Java的互操作性而闻名。

Kotlin 语言特点
  • 简洁的语法，减少样板代码
  • 空安全设计，避免空指针异常
  • 与Java 100%互操作
  • 支持面向对象和函数式编程范式
  • 扩展函数和属性
  • 协程支持异步编程
  • 类型推断，减少显式类型声明
  • 数据类、密封类等现代特性
  • 跨平台开发能力（JVM、JS、Native）
  • 智能转换和类型检查

2. Kotlin 基础语法

Kotlin - 简洁优雅的基础语法

Kotlin的基础语法设计注重简洁性和可读性，同时保持了与Java的互操作性。以下是Kotlin基础语法的核心特性。

Hello World 示例

Kotlin的入口点函数是main()，可以接受命令行参数也可以不接受。

注意事项：Kotlin使用分号作为可选分隔符，通常不需要显式添加分号。

// Hello World 程序
fun main() {
  println("Hello, Kotlin!")
}
// 带参数的 main 函数
fun main(args: Array<String>) {
  println("Hello, Kotlin with args!")
  args.forEach { println("Argument: $it") }
}
变量和常量

Kotlin提供了两种变量声明方式：var（可变）和val（不可变）。推荐优先使用val来提高代码的稳定性和可读性。

注意事项：

  • var声明的变量可以被重新赋值
  • val声明的变量一旦初始化就不能被修改（类似于Java的final）
  • lateinit只能用于var，不能用于val
  • lazy委托只能用于val，不能用于var
// 可变变量 (var)
var age: Int = 30
age = 31 // 可以修改
// 不可变变量 (val) - 推荐使用
val name: String = "Alice"
// name = "Bob" // 编译错误，val 不可修改
// 类型推断
val message = "Hello" // 推断为 String
val count = 42 // 推断为 Int
val price = 19.99 // 推断为 Double
val isActive = true // 推断为 Boolean
// 延迟初始化
lateinit var lateVar: String
lateVar = "Initialized later"
// 惰性求值
val lazyValue: String by lazy {
  println("Computed!")
  "Lazy value"
}
基本数据类型

Kotlin提供了丰富的基本数据类型，包括数字、字符、布尔和字符串。与Java不同，Kotlin的基本数据类型都是对象，可以调用方法。

注意事项：

  • 数字类型有明确的大小限制，避免溢出
  • Long类型需要后缀L，Float类型需要后缀F
  • 字符串模板使用$符号，表达式需要用${}包裹
  • 多行字符串使用三重引号"""，可以保留格式
// 数字类型
val byte: Byte = 127
val short: Short = 32767
val int: Int = 2147483647
val long: Long = 9223372036854775807L // 后缀 L
val float: Float = 3.14F // 后缀 F
val double: Double = 3.141592653589793
// 字符类型
val char: Char = 'A'
val unicodeChar: Char = '\u0041' // Unicode 编码
// 布尔类型
val trueValue: Boolean = true
val falseValue: Boolean = false
// 字符串类型
val string: String = "Hello, Kotlin!"
val multiLineString = """
  This is a
  multi-line
  string
"""
// 字符串模板
val name = "Alice"
val greeting = "Hello, $name!"
val age = 30
val info = "$name is $age years old."
val expression = "${2 + 2} is the result."

3. Kotlin 函数

Kotlin - 强大灵活的函数系统

Kotlin的函数系统非常强大，支持多种函数定义方式和高级特性，如默认参数、命名参数、扩展函数和Lambda表达式等。

函数定义

Kotlin的函数使用fun关键字定义，可以有不同的形式：

注意事项：

  • 函数参数可以有默认值，提高代码复用性
  • 使用命名参数可以提高代码可读性，尤其是在参数较多时
  • 单行函数可以使用表达式函数体简化语法
// 基本函数定义
fun add(a: Int, b: Int): Int {
  return a + b
}
// 表达式函数体（单行函数）
fun subtract(a: Int, b: Int): Int = a - b
// 类型推断的表达式函数体
fun multiply(a: Int, b: Int) = a * b
// 默认参数
fun greet(name: String, message: String = "Hello") {
  println("$message, $name!")
}
// 使用默认参数
fun main() {
  greet("Alice") // 使用默认 message
  greet("Bob", "Hi") // 提供自定义 message
}
// 命名参数
fun format(userName: String, age: Int, email: String) {
  println("Name: $userName, Age: $age, Email: $email")
}
// 使用命名参数
format(userName = "Alice", age = 30, email = "alice@example.com")
format(email = "bob@example.com", userName = "Bob", age = 25) // 可以改变顺序
可变参数和扩展函数

可变参数允许函数接收任意数量的参数，扩展函数允许为现有类型添加新功能。

注意事项：

  • 一个函数只能有一个可变参数，且通常放在参数列表最后
  • 展开数组时需要使用*运算符
  • 扩展函数不能访问私有或受保护的成员
  • 扩展属性需要提供getter和setter实现
// 可变参数
fun sum(vararg numbers: Int): Int {
  var result = 0
  for (number in numbers) {
    result += number
  }
  return result
}
// 使用可变参数
sum(1, 2, 3) // 6
sum(1, 2, 3, 4, 5) // 15
// 展开数组
val array = intArrayOf(1, 2, 3)
sum(*array) // 6
// 扩展函数 - 为现有类型添加新功能
fun String.isPalindrome(): Boolean {
  val clean = this.toLowerCase().replace("\s", "")
  return clean == clean.reversed()
}
// 使用扩展函数
val word = "level"
println(word.isPalindrome()) // true
// 扩展属性
val String.lengthTimesTwo: Int
  get() = this.length * 2
println("hello".lengthTimesTwo) // 10
Lambda 表达式和高阶函数

Lambda表达式是一种简洁的函数表示方式，高阶函数可以接收函数作为参数或返回函数。

注意事项：

  • Lambda表达式的参数类型可以推断，除非编译器无法确定
  • 如果Lambda是函数的最后一个参数，可以放在括号外面
  • 只有一个参数的Lambda可以使用it关键字代替参数名
  • 标准库提供了丰富的高阶函数，如map、filter、reduce等
// Lambda 表达式基本语法
val add = { a: Int, b: Int -> a + b }
val result = add(5, 3) // 8
// 高阶函数 - 接收函数作为参数
fun operate(a: Int, b: Int, operation: (Int, Int) -> Int): Int {
  return operation(a, b)
}
// 使用高阶函数
val sum = operate(5, 3) { x, y -> x + y }
val product = operate(5, 3) { x, y -> x * y }
// 高阶函数 - 返回函数
fun makeMultiplier(factor: Int): (Int) -> Int {
  return { number -> number * factor }
}
// 使用返回函数的高阶函数
val double = makeMultiplier(2)
double(5) // 10
// 标准库中的高阶函数
val numbers = listOf(1, 2, 3, 4, 5)
val doubled = numbers.map { it * 2 } // [2, 4, 6, 8, 10]
val evenNumbers = numbers.filter { it % 2 == 0 } // [2, 4]
val sum = numbers.reduce { acc, number -> acc + number } // 15

4. Kotlin 类和对象

Kotlin - 简洁高效的面向对象编程

Kotlin的面向对象编程支持类、对象、继承、接口等特性，同时提供了数据类、密封类等现代特性，使代码更加简洁高效。

类的定义和实例化

Kotlin的类使用class关键字定义，可以包含属性、方法、初始化块和构造函数。

注意事项：

  • 类可以有一个主要构造函数和多个次要构造函数
  • 初始化块在构造函数执行时被调用
  • 次要构造函数必须调用主要构造函数或其他次要构造函数
// 基本类定义
class Person {
  // 属性
  var name: String = ""
  var age: Int = 0
  
  // 初始化块
  init {
    println("Person initialized")
  }
  
  // 次要构造函数
  constructor(name: String) {
    this.name = name
  }
  
  // 主要构造函数和次要构造函数
  constructor(name: String, age: Int) : this(name) {
    this.age = age
  }
  
  // 方法
  fun greet() {
    println("Hello, my name is $name and I'm $age years old.")
  }
}
// 使用主要构造函数实例化
val person1 = Person("Alice", 30)
person1.greet() // Hello, my name is Alice and I'm 30 years old.
// 使用次要构造函数实例化
val person2 = Person("Bob")
person2.age = 25
person2.greet() // Hello, my name is Bob and I'm 25 years old.
简化的类定义

Kotlin提供了多种简化类定义的方式，如将主要构造函数放在类头中，使用数据类等。

注意事项：

  • 数据类自动生成equals()、hashCode()、toString()和copy()方法
  • 密封类用于受限的类层次结构，只能在同一文件中继承
  • 数据类的主构造函数至少需要一个参数
// 主要构造函数在类头中定义
class Person(val name: String, var age: Int) { fun greet() { println("Hello, my name is $name and I'm $age years old.") } }
// 实例化
val person = Person("Alice", 30)
person.greet()
// 数据类 - 自动生成 equals(), hashCode(), toString(), copy()
data class User(val id: Int, val name: String, val email: String)
val user1 = User(1, "Alice", "alice@example.com")
val user2 = User(1, "Alice", "alice@example.com")
println(user1) // User(id=1, name=Alice, email=alice@example.com)
println(user1 == user2) // true
// 复制数据类
val user3 = user1.copy(email = "alice.new@example.com")
// 密封类 - 用于受限的类层次结构
sealed class Result { data class Success(val data: String) : Result() data class Error(val message: String) : Result() object Loading : Result() }
// 使用密封类
fun handleResult(result: Result) { when (result) { is Result.Success -> println("Success: ${result.data}") is Result.Error -> println("Error: ${result.message}") Result.Loading -> println("Loading...") } }
对象表达式和伴生对象

Kotlin提供了对象表达式和对象声明两种方式来创建单例对象。

注意事项：

  • 对象表达式用于创建匿名类实例
  • 伴生对象是类级别的单例，可以访问类的私有成员
  • 单例对象在第一次被访问时初始化
// 对象表达式 - 匿名类实例
interface Greeter { fun greet() }
val greeter = object : Greeter { override fun greet() { println("Hello from object expression!") } }
greeter.greet()
// 伴生对象 - 类级别的单例
class MyClass { companion object { const val CONSTANT = "Value" fun create(): MyClass { return MyClass() } } }
// 使用伴生对象
val value = MyClass.CONSTANT
val instance = MyClass.create()
// 单例对象
object Singleton { var count = 0 fun increment() { count++ } }
// 使用单例对象
Singleton.increment()
println(Singleton.count) // 1

5. Kotlin 继承和接口

Kotlin - 灵活的继承和接口系统

Kotlin支持继承和接口，同时通过开放类和智能转换等特性，使面向对象编程更加灵活和安全。

继承

Kotlin的类默认是final的，需要使用open关键字才能被继承。子类通过:符号继承父类。

注意事项：

  • 类、方法和属性默认都是final的，需要使用open关键字才能被继承或重写
  • 子类必须调用父类的构造函数
  • Kotlin支持智能转换，无需显式转换类型
  • as?操作符提供安全的类型转换，失败时返回null
// 开放类 - 默认情况下类是 final 的，需要用 open 关键字才能被继承
open class Animal(val name: String) { open fun makeSound() { println("Animal makes sound") } }
// 继承 Animal 类
class Dog(name: String) : Animal(name) { override fun makeSound() { println("Woof!") } }
class Cat(name: String) : Animal(name) { override fun makeSound() { println("Meow!") } }
// 使用继承
val dog = Dog("Buddy")
dog.makeSound() // Woof!
val cat = Cat("Whiskers")
cat.makeSound() // Meow!
// 类型检查和转换
if (dog is Animal) { println("Dog is an animal") }
// 智能转换
if (dog is Dog) { dog.makeSound() // 无需显式转换 }
// 显式转换
val animal: Animal = Dog("Rex")
val dog2 = animal as Dog // 不安全的转换，如果不是 Dog 会抛出异常
val dog3 = animal as? Dog // 安全的转换，如果不是 Dog 会返回 null
接口

Kotlin的接口使用interface关键字定义，可以包含抽象方法和属性，也可以有默认实现。

注意事项：

  • 接口中的方法默认是抽象的，无需显式声明abstract
  • 接口可以有默认方法实现
  • 类可以实现多个接口
  • 接口中的属性必须是抽象的，或者提供getter实现
// 接口定义
interface Shape { val area: Double fun draw() }
// 实现接口
class Rectangle(val width: Double, val height: Double) : Shape { override val area: Double get() = width * height override fun draw() { println("Drawing a rectangle with width $width and height $height") } }
class Circle(val radius: Double) : Shape { override val area: Double get() = Math.PI * radius * radius override fun draw() { println("Drawing a circle with radius $radius") } }
// 使用接口
val shapes: List<Shape> = listOf( Rectangle(5.0, 3.0), Circle(2.5) )
for (shape in shapes) { shape.draw() println("Area: ${shape.area}") }
// 接口中的默认方法
interface Movable { fun move() { println("Moving") } }
class Car : Movable { // 可以覆盖默认方法 override fun move() { println("Car is driving") } }
class Bicycle : Movable { // 使用默认方法 }

6. Kotlin 控制流

Kotlin - 强大的控制流语句

Kotlin提供了丰富的控制流语句，包括if表达式、when表达式（替代switch）、for循环、while循环等，使代码的流程控制更加灵活和简洁。

条件表达式

Kotlin的if是表达式而不是语句，它可以返回值。when表达式是switch的替代，功能更强大。

注意事项：

  • if表达式可以用作三元运算符的替代
  • when表达式可以匹配值、类型、范围等
  • when表达式不需要break，每个分支默认是互斥的
  • when表达式可以有一个else分支处理默认情况
// if 表达式（Kotlin 中 if 是表达式，不是语句）
val a = 10
val b = 20
val max = if (a > b) a else b
// 多行 if 表达式
val result = if (a > b) { println("a is larger") a } else { println("b is larger") b }
// when 表达式（替代 switch）
val day = 3
val dayName = when (day) { 1 -> "Monday" 2 -> "Tuesday" 3 -> "Wednesday" 4 -> "Thursday" 5 -> "Friday" 6 -> "Saturday" 7 -> "Sunday" else -> "Invalid day" }
// when 表达式的其他用法
when { a < 10 -> println("a is less than 10") a < 20 -> println("a is less than 20") else -> println("a is 20 or more") }
// when 用于类型检查
val obj: Any = "Hello"
when (obj) { is String -> println("It's a string: ${obj.length}") is Int -> println("It's an integer: $obj") else -> println("It's something else") }
循环结构

Kotlin支持多种循环结构，包括for循环（用于遍历范围、集合等）、while循环和do-while循环。

注意事项：

  • for循环可以遍历范围、集合、数组等任何可迭代对象
  • 范围使用..操作符创建，如1..5表示1到5（包含）
  • 可以使用downTo创建倒序范围
  • 可以使用step指定步长
  • while和do-while循环与Java类似
// for 循环遍历范围
for (i in 1..5) {
  println(i)
}
// 倒序遍历
for (i in 5 downTo 1) {
  println(i)
}
// 步长
for (i in 1..10 step 2) {
  println(i) // 1, 3, 5, 7, 9
}
// for 循环遍历集合
val numbers = listOf(1, 2, 3, 4, 5)
for (number in numbers) {
  println(number)
}
// 遍历带索引
for ((index, value) in numbers.withIndex()) {
  println("Index $index: $value")
}
// while 循环
var i = 1
while (i <= 5) {
  println(i)
  i++
}
// do-while 循环
var j = 1
do {
  println(j)
  j++
} while (j <= 5)
跳转表达式

Kotlin支持break、continue和return跳转表达式，可以控制程序的执行流程。

注意事项：

  • break用于终止当前循环
  • continue用于跳过当前迭代，进入下一次循环
  • 可以使用标签控制break和continue的作用范围
  • return@label用于从lambda表达式返回，而不是从包含函数返回
// break 和 continue
for (i in 1..10) {
  if (i == 5) break // 终止循环
  println(i)
}
for (i in 1..10) {
  if (i % 2 == 0) continue // 跳过当前迭代
  println(i)
}
// 标签
outer@ for (i in 1..3) {
  for (j in 1..3) {
    if (i * j > 4) break@outer // 跳出外层循环
    println("i = $i, j = $j")
  }
}
// return 表达式
fun findFirstPositive(numbers: List<Int>): Int? {
  for (number in numbers) {
    if (number > 0) return number // 从函数返回
  }
  return null
}
// return@label 从 lambda 返回
val numbers = listOf(1, -2, 3, -4, 5)
numbers.forEach {
  if (it < 0) return@forEach // 从 lambda 返回，不是从函数
  println(it)
}

7. Kotlin 集合

Kotlin - 丰富的集合框架

Kotlin的集合框架提供了丰富的集合类型和操作方法，包括List、Set、Map等，支持不可变和可变两种形式，使集合操作更加安全和高效。

集合类型

Kotlin提供了三种基本的集合类型：List（有序，允许重复）、Set（无序，不允许重复）和Map（键值对）。每种类型都有不可变和可变两种形式。

注意事项：

  • 不可变集合是线程安全的，推荐优先使用
  • 可变集合允许修改，但不是线程安全的
  • setOf()会自动去重重复元素
  • mapOf()使用to关键字创建键值对
// List - 有序集合，允许重复元素
val numbers: List<Int> = listOf(1, 2, 3, 4, 5)
val mutableNumbers: MutableList<Int> = mutableListOf(1, 2, 3)
mutableNumbers.add(4)
// Set - 无序集合，不允许重复元素
val uniqueNumbers: Set<Int> = setOf(1, 2, 3, 2, 1) // {1, 2, 3}
val mutableSet: MutableSet<Int> = mutableSetOf(1, 2, 3)
mutableSet.add(4)
// Map - 键值对集合
val map: Map<String, Int> = mapOf("one" to 1, "two" to 2, "three" to 3)
val mutableMap: MutableMap<String, Int> = mutableMapOf("one" to 1)
mutableMap["two"] = 2
// 空安全集合
val emptyList: List<Int> = emptyList()
val emptySet: Set<String> = emptySet()
val emptyMap: Map<String, Int> = emptyMap()
// 类型推断的集合
val inferredList = listOf(1, 2, 3) // List<Int>
val inferredMap = mapOf("a" to 1, "b" to 2) // Map<String, Int>
集合操作

Kotlin的集合提供了丰富的操作方法，如过滤、映射、排序、分组等，支持函数式编程风格。

注意事项：

  • 不可变集合的操作会返回新的集合，不会修改原集合
  • 可变集合的操作可以修改原集合
  • reduce()和fold()的区别是fold()有初始值
  • first()如果没有匹配元素会抛出异常，firstOrNull()会返回null
val numbers = listOf(1, 2, 3, 4, 5)
// 过滤
val evenNumbers = numbers.filter { it % 2 == 0 } // [2, 4]
val oddNumbers = numbers.filterNot { it % 2 == 0 } // [1, 3, 5]
val greaterThanTwo = numbers.filter { it > 2 } // [3, 4, 5]
// 映射
val doubled = numbers.map { it * 2 } // [2, 4, 6, 8, 10]
val squared = numbers.map { it * it } // [1, 4, 9, 16, 25]
// 扁平化
val nestedList = listOf(listOf(1, 2), listOf(3, 4), listOf(5))
val flattened = nestedList.flatten() // [1, 2, 3, 4, 5]
val flatMapped = nestedList.flatMap { it.map { it * 2 } } // [2, 4, 6, 8, 10]
// 折叠和归约
val sum = numbers.reduce { acc, number -> acc + number } // 15
val sumWithInitial = numbers.fold(10) { acc, number -> acc + number } // 25
// 排序
val unsorted = listOf(3, 1, 4, 1, 5, 9, 2, 6)
val sorted = unsorted.sorted() // [1, 1, 2, 3, 4, 5, 6, 9]
val sortedDescending = unsorted.sortedDescending() // [9, 6, 5, 4, 3, 2, 1, 1]
val sortedByLength = listOf("apple", "banana", "cherry").sortedBy { it.length } // [apple, cherry, banana]
// 分组
val words = listOf("apple", "banana", "cherry", "date", "elderberry")
val groupedByLength = words.groupBy { it.length } // {5=[apple], 6=[banana, cherry], 4=[date], 10=[elderberry]}
// 元素检查
val hasEven = numbers.any { it % 2 == 0 } // true
val allPositive = numbers.all { it > 0 } // true
val noneNegative = numbers.none { it < 0 } // true
val firstEven = numbers.first { it % 2 == 0 } // 2
val lastEven = numbers.last { it % 2 == 0 } // 4
val firstOrNull = numbers.firstOrNull { it > 10 } // null
集合转换

Kotlin的集合提供了多种转换方法，可以将集合转换为其他类型或格式。

注意事项：

  • toTypedArray()将集合转换为数组
  • joinToString()可以自定义分隔符、前缀和后缀
  • associateWith()将集合元素作为键，生成的值作为值
  • zip()可以将两个集合合并为键值对
val numbers = listOf(1, 2, 3, 4, 5)
// 转换为其他集合类型
val array = numbers.toTypedArray() // Array<Int>
val set = numbers.toSet() // Set<Int>
val hashSet = numbers.toHashSet() // HashSet<Int>
// 转换为字符串
val joined = numbers.joinToString() // "1, 2, 3, 4, 5"
val customJoined = numbers.joinToString(separator = "|", prefix = "[", postfix = "]") // "[1|2|3|4|5]"
// 映射到其他类型
val stringList = numbers.map { it.toString() } // List<String>
val mapFromList = numbers.associateWith { it * 2 } // Map<Int, Int> {1=2, 2=4, 3=6, 4=8, 5=10}
val mapFromPairs = numbers.zip(numbers.map { it * 2 }).toMap() // Map<Int, Int> {1=2, 2=4, 3=6, 4=8, 5=10}
// 过滤并转换
val evenSquares = numbers.filter { it % 2 == 0 }.map { it * it } // [4, 16]

8. Kotlin 异常处理

Kotlin - 安全的异常处理机制

Kotlin的异常处理机制与Java类似，但提供了空安全特性来避免空指针异常，使代码更加安全可靠。

抛出和捕获异常

Kotlin使用throw关键字抛出异常，使用try-catch-finally块捕获和处理异常。

注意事项：

  • try可以作为表达式使用，返回值是try块或catch块的最后一个表达式
  • finally块无论是否发生异常都会执行
  • 可以自定义异常类继承Exception或RuntimeException
  • Kotlin不区分检查异常和非检查异常
// 抛出异常
fun divide(a: Int, b: Int): Int {
  if (b == 0) {
    throw ArithmeticException("Division by zero")
  }
  return a / b
}
// 捕获异常
try {
  val result = divide(10, 0)
  println("Result: $result")
} catch (e: ArithmeticException) {
  println("Error: ${e.message}")
} catch (e: Exception) {
  println("Unexpected error: ${e.message}")
} finally {
  println("This block always executes")
}
// try 作为表达式
val result = try {
  divide(10, 2)
} catch (e: ArithmeticException) {
  0 // 默认值
}
// 自定义异常
class InvalidInputException(message: String) : Exception(message)
fun processInput(input: String) {
  if (input.isEmpty()) {
    throw InvalidInputException("Input cannot be empty")
  }
  // 处理输入
}
空安全和异常

Kotlin的空安全特性通过类型系统来避免空指针异常，提供了多种操作符来安全地处理可能为空的值。

注意事项：

  • 安全调用操作符?.在对象不为空时调用方法或属性，否则返回null
  • Elvis操作符?:在左侧表达式为null时使用右侧的默认值
  • 非空断言操作符!!强制断言对象不为空，为空时会抛出异常，不推荐使用
  • 安全转换操作符as?在转换失败时返回null，而不是抛出异常
  • let函数与安全调用操作符结合使用，可以在对象不为空时执行代码块
// 安全调用操作符 ?.
val str: String? = null
val length = str?.length // null
// Elvis 操作符 ?:
val safeLength = str?.length ?: 0 // 0
// 非空断言操作符 !!.（不推荐使用）
val unsafeLength = str!!.length // 抛出 NullPointerException
// 安全转换操作符 as?
val obj: Any = "Hello"
val string: String? = obj as? String
val number: Int? = obj as? Int // null
// let 函数与空安全
str?.let {
  println("Length: ${it.length}") // 只有当 str 不为 null 时执行
}
// 空安全类型参数
fun <T: Any> printLength(obj: T?) {
  obj?.let {
    println("Length: ${it.toString().length}")
  }
}

9. Kotlin 协程

Kotlin - 轻量级的协程并发编程

Kotlin的协程是一种轻量级的并发编程解决方案，允许以顺序代码的方式编写异步代码，使并发编程更加简单和安全。

协程基础

协程是可以暂停和恢复的计算过程，比线程更轻量级，可以创建大量协程而不会消耗太多资源。

注意事项：

  • 协程需要添加kotlinx-coroutines-core依赖
  • runBlocking会阻塞当前线程，主要用于测试和main函数
  • launch启动一个新的协程，不阻塞当前线程
  • delay是一个挂起函数，不会阻塞线程
// 协程依赖
// implementation 'org.jetbrains.kotlinx:kotlinx-coroutines-core:1.7.3'
import
kotlinx.coroutines.*
// 简单的协程
fun main() = runBlocking { // 这会阻塞主线程，直到协程完成
  launch { // 在后台启动一个新的协程
    delay(1000L) // 非阻塞延迟 1 秒
    println("World!")
  }
  println("Hello") // 立即执行
}
// 等待协程完成
fun main() = runBlocking {
  val job = launch {
    delay(1000L)
    println("World!")
  }
  println("Hello")
  job.join() // 等待协程完成
  println("Done")
}
// 协程作用域
fun main() = runBlocking {
  val scope = CoroutineScope(Dispatchers.Default)
  val job = scope.launch {
    delay(1000L)
    println("World!")
  }
  println("Hello")
  job.join()
}
协程上下文和调度器

协程上下文包含协程的运行环境信息，调度器决定协程在哪个线程上执行。

注意事项：

  • Dispatchers.Main：在主线程运行，主要用于UI操作
  • Dispatchers.IO：用于IO操作，如网络请求、文件读写
  • Dispatchers.Default：用于CPU密集型操作
  • Dispatchers.Unconfined：不受限的调度器，在当前线程执行
  • NewSingleThreadContext：创建一个新的单线程上下文
import kotlinx.coroutines.*
fun main() = runBlocking {
  // 默认调度器（与调用者相同的线程）
  launch {
    println("Default: ${Thread.currentThread().name}")
  }
  
  // 主线程调度器
  launch(Dispatchers.Main) {
    println("Main: ${Thread.currentThread().name}")
  }
  
  // IO 调度器（用于网络、文件操作等）
  launch(Dispatchers.IO) {
    println("IO: ${Thread.currentThread().name}")
  }
  
  // 默认调度器（线程池）
  launch(Dispatchers.Default) {
    println("Default: ${Thread.currentThread().name}")
  }
  
  // 新线程调度器
  launch(Dispatchers.NewSingleThreadContext("MyThread")) {
    println("NewSingleThreadContext: ${Thread.currentThread().name}")
  }
}
// 切换调度器
fun main() = runBlocking {
  launch(Dispatchers.Default) {
    println("Starting in ${Thread.currentThread().name}")
    withContext(Dispatchers.IO) {
      println("Working in ${Thread.currentThread().name}")
    }
    println("Back to ${Thread.currentThread().name}")
  }
}
异步编程和协程

协程可以简化异步编程，使异步代码看起来像同步代码一样易于理解和维护。

注意事项：

  • suspend关键字标记的函数只能在协程或其他挂起函数中调用
  • async启动一个新协程并返回一个Deferred对象，可以通过await()获取结果
  • async(start = CoroutineStart.LAZY)创建惰性协程，只有在需要时才会执行
  • 并发执行可以提高程序的性能，但要注意线程安全问题
import kotlinx.coroutines.*
import kotlin.system.measureTimeMillis
// 异步函数
suspend fun doSomethingUsefulOne(): Int {
  delay(1000L)
  return 13
}
suspend fun doSomethingUsefulTwo(): Int {
  delay(1000L)
  return 29
}
// 顺序执行
fun main() = runBlocking {
  val time = measureTimeMillis {
    val one = doSomethingUsefulOne()
    val two = doSomethingUsefulTwo()
    println("The answer is ${one + two}")
  }
  println("Completed in $time ms") // ~2000 ms
}
// 并发执行
fun main() = runBlocking {
  val time = measureTimeMillis {
    val one = async { doSomethingUsefulOne() }
    val two = async { doSomethingUsefulTwo() }
    println("The answer is ${one.await() + two.await()}")
  }
  println("Completed in $time ms") // ~1000 ms
}
// 惰性异步
fun main() = runBlocking {
  val time = measureTimeMillis {
    val one = async(start = CoroutineStart.LAZY) { doSomethingUsefulOne() }
    val two = async(start = CoroutineStart.LAZY) { doSomethingUsefulTwo() }
    // 启动协程
    one.start()
    two.start()
    println("The answer is ${one.await() + two.await()}")
  }
  println("Completed in $time ms") // ~1000 ms
}

10. Kotlin 高级特性

Kotlin - 强大的高级特性

Kotlin提供了多种高级特性，如委托、泛型、反射和Java互操作等，使代码更加灵活和强大。

委托

委托是一种设计模式，Kotlin内置了对委托的支持，包括类委托和属性委托。

注意事项：

  • 类委托使用by关键字将接口实现委托给另一个对象
  • 属性委托可以实现延迟加载、可观察属性等功能
  • lazy委托用于延迟初始化属性，只有在第一次访问时才会计算
  • Delegates.observable用于监听属性值的变化
  • 映射委托可以将属性值存储在Map中
// 委托模式
interface Base {
  fun print()
}
class BaseImpl(val x: Int) : Base {
  override fun print() {
    println(x)
  }
}
// 委托给 BaseImpl
class Derived(b: Base) : Base by b
// 使用委托
val base = BaseImpl(10)
val derived = Derived(base)
derived.print() // 10
// 属性委托
class Example {
  // 延迟属性
  val lazyValue: String by lazy {
    println("Computed!")
    "Hello"
  }
  
  // 可观察属性
  var observableValue: String by Delegates.observable("Initial value") { _, old, new ->
    println("Changed from $old to $new")
  }
  
  // 映射委托
  val map = mapOf(
    "name" to "Alice",
    "age" to 30
  )
  val name: String by map
  val age: Int by map
}
val example = Example()
println(example.lazyValue) // 打印 "Computed!" 和 "Hello"
example.observableValue = "New value" // 打印 "Changed from Initial value to New value"
println(example.name) // "Alice"
println(example.age) // 30
泛型

泛型允许在定义类、函数和接口时使用类型参数，使代码更加通用和可复用。

注意事项：

  • 泛型类使用<T>语法定义类型参数
  • 泛型函数可以在函数名前声明类型参数
  • 泛型约束可以限制类型参数的范围
  • 星投影<*>用于表示未知类型参数
  • Kotlin的泛型是实化的，可以在运行时获取类型信息
// 泛型类
class Box<T>(val value: T) {
  fun getValue(): T {
    return value
  }
}
// 使用泛型类
val intBox = Box(10)
val stringBox = Box("Hello")
val intValue: Int = intBox.getValue()
val stringValue: String = stringBox.getValue()
// 泛型函数
fun <T> printValue(value: T) {
  println(value)
}
// 使用泛型函数
printValue(10)
printValue("Hello")
// 泛型约束
fun <T : Comparable<T>> max(first: T, second: T): T {
  return if (first > second) first else second
}
// 使用带约束的泛型函数
val maxInt = max(10, 20) // 20
val maxString = max("apple", "banana") // "banana"
// 星投影
val boxes: List<Box<*>> = listOf(intBox, stringBox)
反射

反射允许在运行时获取类的信息、调用方法和访问属性，是一种强大的元编程技术。

注意事项：

  • 反射需要添加kotlin-reflect依赖
  • 使用::class获取类的KClass对象
  • 可以获取构造函数、属性和方法的信息
  • 反射操作可能会影响性能，应谨慎使用
  • 反射可以突破访问控制，访问私有成员
// 需要添加反射依赖
// implementation 'org.jetbrains.kotlin:kotlin-reflect:1.9.20'
import kotlin.reflect.full.memberProperties
import kotlin.reflect.full.primaryConstructor
class Person(val name: String, val age: Int) {
  fun greet() {
    println("Hello, my name is $name")
  }
}
// 获取类的 KClass
val personClass = Person::class
// 获取主构造函数
val constructor = personClass.primaryConstructor
val person = constructor?.call("Alice", 30)
// 获取属性
for (property in personClass.memberProperties) {
  println("${property.name} = ${property.get(person)}")
}
// 调用方法
val greetMethod = personClass.members.find { it.name == "greet" }
greetMethod?.call(person)

Kotlin与Java有很好的互操作性，可以在Kotlin中使用Java类，也可以在Java中使用Kotlin类。

内联函数

内联函数允许将函数的字节码直接插入到调用点，减少函数调用的开销，特别适用于高阶函数。

注意事项：

  • 使用inline关键字声明内联函数
  • 内联函数会增加生成的字节码大小
  • crossinline关键字用于禁止在Lambda表达式中使用非局部返回
  • noinline关键字用于防止某些参数被内联
  • 内联函数适用于简单的高阶函数，复杂函数不适合内联
// 内联函数 - 减少函数调用开销
inline fun measureTime(block: () -> Unit) {
  val start = System.nanoTime()
  block()
  val end = System.nanoTime()
  println("Execution time: ${(end - start) / 1000000} ms")
}
// 使用内联函数
fun main() {
  measureTime {
    for (i in 1..1000000) {
      val sum = i + i
    }
  }
}
// crossinline - 禁止非局部返回
inline fun inlineFunction(crossinline block: () -> Unit) {
  println("Before block")
  block()
  println("After block")
}
// noinline - 防止某些参数被内联
inline fun mixedInlineFunction(inlineBlock: () -> Unit, noinline nonInlineBlock: () -> Unit) {
  inlineBlock() // 会被内联
  nonInlineBlock() // 不会被内联
}

Kotlin提供了类型别名、运算符重载和中缀函数等特性，使代码更加灵活和易读。

注意事项：

  operator fun plus(other: Point): Point {

  }

    return Point(x * factor, y * factor)
  }

      0 -> x
      1 -> y
